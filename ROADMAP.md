# Kiwi Development Roadmap
## Stage 1 - Core Systems:
In this stage I plan on setting up modular systems such as main kiwi striker tool, and a simple way for anyone to add their own tools

## Stage 2 - Building
Here Kiwi will become a more casual project. Anyone can upload their own exploits, develop their own tools within the kiwi repository.
I would encourage anybody, once the basics of kiwi is setup, to add your own folder with your own tools to the bin folder.

## Stage 3 - Advanced Features
Adding extra features possibly including:
- Simple scripting language for automation
- Log scraping tools for searching through large log files
- Configuration files for each tool
